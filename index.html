NJ Colleges And Map
Android App

This Android app, developed in Kotlin, showcases three fragments to provide comprehensive information about colleges in New Jersey. The first fragment features a list of NJ colleges with names and addresses. The second fragment displays all the colleges on a Google Map, utilizing the Google Maps API and markers. The app requests permission to display the user’s location as well as calculate the user's distance from each college - displaying this information on each marker. The third fragment presents a summary, including a count of colleges in each region. Features such as a compass, zooming options, and descriptive icons are included. This app seamlessly combines mapping, listing, and summarizing functionalities to offer an immersive experience for users exploring colleges in New Jersey.

NJ Gas Pumps
Android App

This Kotlin-based Android app serves as a comprehensive solution for accessing gas station information in New Jersey. By parsing a JSON file, the app displays a list of gas stations with details such as their names, logos, towns, and gas prices. Upon selecting a gas station from the list, users are presented with a detailed view showcasing the street address, town, state, and zip code, along with the station's logo. Additionally, the app includes a user-friendly section for updating gas prices. With its intuitive interface and ability to provide real-time information, this app offers a valuable tool for individuals seeking accurate gas station data and convenient price updates.

Euro Calculator
Android App

This Android app was developed using Kotlin in Android Studio. It serves as a user-friendly currency converter, allowing users to input a dollar amount and swiftly convert it to euros. The default exchange rate of 0.98120 ensures immediate results, but users also have the flexibility to set their own exchange rate based on the current dollar to euro exchange rate. The sleek and intuitive interface offers a text field for dollar input, a "Convert" button, a display of the converted euro amount, a text field for exchange rate input, and a “Set Rate Button”. This app is a valuable tool for quick and accurate currency conversion, benefiting both individuals and businesses.

Drop Game
Java

Snake Game
Java

The Snake Game, coded in Java, offers an exciting and engaging experience with added features to enhance gameplay. Users can choose between easy or hard mode, with hard mode presenting a challenge by removing the grid and increasing the speed of the snake. In this classic game, players control the snake to navigate the screen and devour apples. A score is prominently displayed at the top of the game, keeping users motivated to achieve high scores. However, if the snake collides with itself or hits a border, a game over screen is triggered, accompanied by the final score. This Java-based Snake Game delivers an enjoyable and addictive gaming experience while providing different difficulty levels and ensuring a thrilling competition for the highest score.
